✈️ تيليغرام
بوت يرسل تفاصيل كل طلب جديد للمجموعة وللمستخدمين المعنيين مباشرة
كيف أحصل على Bot Token و Chat ID؟
  1. 1
    افتح تيليغرام وابحث عن @BotFather
  2. 2
    أرسل /newbot واختر اسماً للبوت
  3. 3
    ستحصل على Bot Token بالشكل: 123456789:AAF...
  4. 4
    أضف البوت لمجموعتك كـ Admin
  5. 5
    افتح رابط api.telegram.org/bot[TOKEN]/getUpdates وابحث عن chat.id

بدون + ، مثال: 9647700000000
اختبار
📧 البريد الإلكتروني (EmailJS)
إيميل مفصّل بتفاصيل الطلب عبر خدمة EmailJS المجانية
كيف أحصل على مفاتيح EmailJS؟
  1. 1
    سجّل مجاناً في emailjs.com
  2. 2
    أنشئ خدمة Gmail/Outlook → ستحصل على Service ID
  3. 3
    أنشئ قالب إيميل → ستحصل على Template ID
  4. 4
    من Account → API Keys → ستجد Public Key
📲 إشعارات الجوال (Push)
غير مفعّل
إشعارات تصل على الجوال حتى لو المتصفح مغلق — تعمل على Android و iOS وأجهزة الكمبيوتر
من Firebase Console → Project Settings → Cloud Messaging → Web Push certificates → Generate key pair
من Firebase Console → Project Settings → Cloud Messaging → Server Key
خطوات تفعيل الإشعارات على جهازك
  1. 1
    احفظ VAPID Key و FCM Server Key من Firebase ثم اضغط "حفظ" أدناه
  2. 2
    سجّل دخولك على المتجر ثم وافق على طلب الإذن بالإشعارات
  3. 3
    سيُحفظ FCM Token الخاص بجهازك تلقائياً
  4. 4
    من الآن ستصلك إشعارات الطلبات على هاتفك حتى لو أغلقت المتصفح
إعداد Firebase Console
  1. 1
    افتح console.firebase.google.com → اختر مشروعك
  2. 2
    Project Settings → Cloud Messaging
  3. 3
    قسم Web Push certificates → Generate key pair → انسخ الـ Key
  4. 4
    قسم Cloud Messaging API (Legacy) → Server Key → انسخه
